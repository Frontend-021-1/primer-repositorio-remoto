# Guía de trabajo en clase

## Paso a paso para trabajar con Git

1. Crear archivo prueba.txt
2. Agregar texto a prueba.txt
3. Inicializar repositorio `git init`
4. Agregar al área de preparación (staging area) `git add .`
5. Agregar al repositorio `git commit -m "mensaje descriptivo"`
6. Hacer cambios en el archivo
7. `git add .` y `git commit -m "mensaje"`
